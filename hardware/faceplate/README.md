# Corrode Faceplate

Custom faceplate overlay for the Daisy Patch Init (10HP).

## Files

- `corrode_faceplate.svg` — Vector source (Inkscape)
- `corrode_faceplate.pdf` — Print-ready version

## Printing

The faceplate is designed as a label/overlay that attaches to the existing Patch Init front panel. Print on adhesive-backed vinyl or matte photo paper at 100% scale.

Dimensions: 10HP = 50.5mm wide × 128.5mm tall
